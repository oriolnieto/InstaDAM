# InstaDAM - Grupo 5

## Descripción del proyecto

InstaDAM es una réplica de la conocida aplicación Instagram.

## Instalación

  cupertino_icons: ^1.0.8
  sqflite: ^2.4.2
  shared_preferences: ^2.5.4
